# 🌤 Weather Dashboard (React + Open Meteo)

A modern, responsive **weather dashboard** built using **React (Vite + TypeScript)** that fetches real-time weather data from the Open Meteo API and displays it with dynamic, theme-based UI.

---

## 🚀 Features

- 🌍 Fetch weather data using latitude, longitude, and date
- 🎨 Dynamic theme system based on weather conditions
- 📊 Interactive charts (temperature & precipitation)
- 📱 Fully responsive design
- ⚡ Optimized API handling using React Query
- 🧪 Unit testing with Vitest + React Testing Library

---

## 🛠 Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Recharts
- TanStack React Query
- Vitest + Testing Library

---

## ⚙️ Getting Started

### 1. Clone the repository

git clone https://github.com/shreyas4510/weather-forecast.git
cd weather-forecast

### 2. Install dependencies

npm install

### 3. Run the development server

npm run dev

App runs at: http://localhost:5173

---

## 🧪 Running Tests

npm run test
